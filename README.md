# glitter

## Installation

```bash
sudo make install
```

or place it in `~/.local/bin` with:
```bash
make local_install
```

## Usage

```
Usage: glitter <COMMAND>

Commands:
  push    Stage, commit, and push changes
  commit  Stage all files and commit
  add     Stage files
  pull    Force pull and reset local changes
  open    Open the project in the default web browser
  help    Print this message or the help of the given subcommand(s)

Options:
```
